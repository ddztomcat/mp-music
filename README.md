# mp-music

> 利用美团开源的MpVue框架，将之前学习的vue项目稍加改造，官方文档链接[MpVue](http://mpvue.com/)

## 项目启动
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
